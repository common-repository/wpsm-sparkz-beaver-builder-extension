== Installation ==
1. Download the latest zip file and extract the `wpsm-beaver-builder-extension` directory.
2. Upload it to your `/wp-content/plugins/` directory.
3. Activate `WPSM Beaver Builder Extension` on the `Plugins` menu in WordPress.

Note : 1. First install Beaver Builder Plugin. <a href="https://wordpress.org/plugins/beaver-builder-lite-version/">Here</a> is link.</br>
